# Comutation Plot

## Data

* TSV of siginificant genes (ex. `SigGenes_001.txt`)
    * Required columns: `gene`, `p`
* TSV of gene mutation information (`genenonsilent200.txt`)

## Plots

### Mutation Type Plot

Mutation types:

* C:G transitions
* C:G transversions
* A:T transitions
* A:T transversions
* null+indel mutations
